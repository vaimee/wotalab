Counter - Semaphore Orchestrator Viewer

This is the visualization for the Semaphore Counter practical lesson. It exposes no thing, but it consumes two remote things:

  • A Counter Thing that maintains a count value and allows incrementing it.
  • A Semaphore Thing that changes its color based on the count value.

The orchestrator periodically increments the counter and updates the semaphore color based on the current count.
